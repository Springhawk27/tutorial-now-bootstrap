Flex or Flexbox layout is a css layout which provides a clean and simple way to arrange items or contents within the container. It has the ability to best fit the items to its container. Flexbox layout is a one dimensional layout and it’s either a row-based or a column-based layout.
On the other hand, Grid layout also provides a clean and simple way to arrange items or contents within the container. But it's a two dimensional layout and Its’ both row and column-based layout.

Bootstrap is a set of pre-styled responsive, mobile-first component-based framework, means it comes with prebuilt components and includes other utilities for layering displays, spacing, etc.
On the other hand, Tailwind CSS is a utility-first framework. Using Tailwind CSS is kind of writing regular CSS with lots of classes which gives much freedom to design the UI. Unlike Bootstrap, it has no prebuilt components.

A box model describes a box where different parts of the box work together to create a layout which we can see in the webpage. Different elements of a box model are content, padding, border and margin.
Content: This is the area where the item is shown.
Padding: Padding is the white space around the content.
Border: It wraps up the content and any padding with it.
Margin: It wraps up the content, padding and border. It is the most outer layer of the box.

Semantic tags in html simply means tags which is meaningful and relevant to the content in it. A semantic tag in html express its meaning to both the developer and the browser.
Some of the semantic tags in html are header, footer, nav, article, form, table, details, section etc. These tags hold some meaning itself. For example, an article tag express that the content inside is about an article. It helps a developer to get the idea of that content easily.
